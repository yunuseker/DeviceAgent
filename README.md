# device information collector
This library helps you easily get Application, Battery, Hardware, Sensor and Wifi information on your Android device. But it is not finished yet.

### Battery Data
```java
BatteryInfo {
    Integer scale;
    String technology;
    Integer level;
    Integer voltage;
    String status;
    String plugged;
    String health;
    Integer temperature;
    }
```
### Hardware Data
```java
HardwareInfo {
    String product;
    String model;
    String osVersion;
    String hardware;
    String manufacturer;
    String device;
    String brand;
    String display;
    String bootloader;
    String board;
    }
```
